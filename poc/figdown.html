FigDown PoC — deterministic text→SVG, draft v0.0

Figures as text — one source, two readers. Edit left, render right. Same source → same SVG, bit-level.

Direct editing (every action rewrites the text): drag a node → pin line · double-click → rename · click to select, then pick a fill color → color= · Raise/Lower → moves the node's line in the text (line order = paint order = implicit layer) · Link → click two nodes → edge line · ⬚ Group → drag a marquee → group + in= · click a table cell → cell r,c color= · drag a threshold line → rewrites its at=% · Pin all → freeze layout so future edits stay local.
