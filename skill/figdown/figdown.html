FigDown PoC — deterministic text→SVG, draft v0.0

Figures as text — one source, two readers. Edit left, render right. Same source → same SVG, bit-level.

Every GUI action rewrites the text: double-click empty canvas → new node + pin · drag a node → pin · arrow keys nudge (Shift = ×10) · drag the corner handle → size · double-click a node → rename · Del → remove node + its pins/edges · fill color → color= · Raise/Lower → line order = paint order · Link → edge · ⬚ Group → group + in= · click a table cell → cell r,c color= · drag a threshold line → at=% · click an error → jump to that line · Ctrl+S save · Ctrl+O open · unsaved work survives in autosave.
